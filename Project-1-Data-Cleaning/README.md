# 📊 Project 1: Data Cleaning & Preparation

## 🎯 Objective
Transform a raw e-commerce dataset into a clean and structured dataset ready for analysis.

---

## 🛠️ Tech Stack
`Python` · `Pandas` · `Jupyter Notebook`

---

## ⚙️ Workflow

- Handled missing values in categorical columns
- Standardized date formats (ISO 8601)
- Removed duplicates and inconsistencies in IDs
- Validated data integrity across all fields

---

## 📦 Outputs

- Cleaned dataset (`cleaned_data.xlsx`)
- Data quality report / change log

---

## 📌 Result
Produced a clean, reliable dataset with consistent formatting and no identifier issues, ready for further analysis.
